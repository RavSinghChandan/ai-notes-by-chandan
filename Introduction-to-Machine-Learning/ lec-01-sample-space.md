# 📌 Topic: Probability Basics

## 🔹 Core Idea
- Sample Space = All possible outcomes
- Event = Subset of outcomes
- Probability = Likelihood of event

## 🔹 Key Concepts
- Union (A ∪ B) → A or B
- Intersection (A ∩ B) → A and B
- Complement (Aᶜ) → Not A

## 🔹 Key Formula
- P(A|B) = P(A ∩ B) / P(B)

## 🔹 Bayes Theorem
- Used to reverse probability
- Important in ML models

## 🔹 Example
Coin toss → {HH, HT, TH, TT}
Event: At least one head

## 🔹 ML Mapping
- Sample Space → All predictions
- Event → Output class
- Probability → Confidence score

- What is sample space?
- What is conditional probability?
